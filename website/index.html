Algorithmic Customization

The bot supports algorithmic customization via a powerful Domain Specific Language (DSL). Users can input custom pseudo-code in the application wizard to override the default multi-timeframe strategy.

custom_strategy.dsl 
IF RSI < 45 AND EMA_TREND == BULLISH THEN BUY
IF RSI > 55 AND EMA_TREND == BEARISH THEN SELL

Impenetrable Risk Controls

Jiggon is inherently designed to protect principal capital through multiple systemic gates.

  • Maximum Daily Drawdown

    Automatically halts execution permanently for the session if realized losses breach the predefined percentage threshold.

  • Take-Profit Halting

    Engages Safe Mode automatically upon reaching nominal profit targets to preserve realized gains.

  • Abnormal Volatility Rejection

    Constantly monitors Average True Range (ATR). Suspends trading if market conditions become erratic.

Frequently Asked Questions

Do I need real money to test this?

No. Jiggon fully supports Deriv's Demo Environment. You can safely test custom algorithms without any financial risk.

What markets are supported?

Currently, Jiggon specializes in Synthetic Volatility Indices (e.g. Vol 10, Vol 100), operating 24/7 without weekend closures.

Is my API token secure?

Yes. Your token is stored locally in jiggon_config.json and is only ever transmitted directly to the Deriv WebSocket API.

Global Installation

Jiggon is distributed as a lightweight Python package. Ensure you have Python 3.10+ installed, then run the following command in your terminal:

pip install jiggon Copy

After installation, simply type jiggon anywhere in your terminal to launch the trading bot.
